Crear fichero de logs en bash

2021-06-09 · 1 min · @Programación · #Debian #Bash

A veces tienes la necesidad de crear un fichero de log para saber cuando se ejecuta un proceso en un script que puedes haber creado, aquí pongo el que yo uso.

A través del siguiente código de bash, podemos crear nuestro propio fichero de logs:

tie=/tmp/script_final.log*
# PER SAPIGUER LA DATA D'INICI*
hi=`date`*
# PER SAPIGUER LA DATA FINAL*
hf=`date`*

echo $hf" " >> $tie*

Con esto ya tenemos nuestro log
